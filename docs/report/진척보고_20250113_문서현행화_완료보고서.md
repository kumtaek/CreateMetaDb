2. **5단계 JSP 분석**

## 🚨 현재 제한사항

1. **4단계 Java 분석 비활성화**
   - `JavaLoadingEngine` 클래스는 완전 구현되어 있으나 main.py에서 주석 처리됨
   - Java 파일 분석 기능을 사용하려면 main.py 수정 필요

2. **5단계 JSP 분석 미구현**
   - JSP 파서 파일 자체가 존재하지 않음
   - JSP → Java 메서드 관계 분석 불가

3. **SQL Content 저장 비활성화**
   - `SqlContentManager`가 주석 처리되어 SQL 내용 압축 저장 기능 미사용
