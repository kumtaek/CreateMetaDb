# Oracle SQL 파서용 메뉴얼

## 개요
Oracle 데이터베이스 SQL 분석을 위한 공식 참조 문서를 저장하는 폴더입니다.

## 보유 메뉴얼
- `sql-language-reference.pdf` - Oracle SQL 언어 참조

## 추가 필요 메뉴얼
- PL/SQL User's Guide and Reference
- Oracle Database SQL Language Reference
- Oracle Database Concepts

## 파서 활용
- SQL 쿼리 분석
- DDL/DML/DCL 구문 분석
- 함수 및 프로시저 분석
- 테이블 및 인덱스 분석
- 제약조건 분석
